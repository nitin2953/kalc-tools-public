# kalc-tools-public

Welcome to the public feedback and support repository for Kalc Tools!

This repository is for:
- Bug reports related to calculators and tools
- Requests for new features or calculator/tools ideas
- General/casual questions and feedback (not legal or copyright related)

**For legal, copyright, or other "serious" questions, please use the contact form at our main website.**

Feel free to participate by creating an issue using the appropriate template!
